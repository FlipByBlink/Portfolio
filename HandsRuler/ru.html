HandsRuler

Интуитивно понятное приложение для измерения расстояния до объекта двумя руками.

Для Apple Vision Pro.

Обзор.

Интуитивно понятное измерение расстояний с помощью обеих рук!

Отображает расстояние между кончиками указательных пальцев обеих рук.

Варианты единиц измерения

  • Сантиметры (см)

  • Метр (м)

  • Дюйм (дюймы)

  • Футы (ft)

  • Ярд (yd)

  • Футы и дюймы

Дополнительные функции

  • Косвенное нажатие на указатель для фиксации положения.

  • Функция многократного измерения.

  • Функция ведения журнала.
