MemorizeWidget

Turn Your iPhone/iPad/Apple Watch/Mac into a Memory Book

Overview

Display your memorandum book on your home screen or lock screen using the widget function. App for iPhone, iPad, Apple Watch, and Mac.

  • Two modes: "stand-alone mode" and "random mode.
  • In "Random Mode," the notes displayed change every 5 minutes
  • Memorandum data can be imported from text or files (csv, tsv, txt, etc.)
  • Linkage with OS built-in dictionary
  • Customizable search function
  • Synchronizes data between devices via iCloud

Multi-platform support

  • iPhone: Home screen, Lock screen, Standby (iOS 17)
  • iPad: Home screen, Lock screen (iPadOS 17)
  • Mac: Notification center, desktop (macOS 14)
  • Apple Watch: Complication, Smart Stack (watchOS 10)

Options

  • Display note "title" as well as "comment" on widget
  • Display multiple notes
