FlipByBlink

Una aplicación para libros electrónicos que te permite pasar páginas con sólo parpadear

Visión general.

Sólo una función especial. Se trata de una aplicación para iPhone y iPad que te permite pasar páginas parpadeando con los ojos cerrados durante un periodo de tiempo ligeramente más largo de forma consciente.

Aparte de las funciones anteriores, se trata de una aplicación de lectura de libros electrónicos sencilla y común.

Cómo funciona el paso de página

Un párpado cerrado ligeramente prolongado hace que la página se envíe hacia delante, y cuando abres los ojos, ya has pasado a la página siguiente. El parpadeo normal no provoca el paso de página.

El parpadeo es detectado por la cámara y los sensores integrados en el dispositivo.

Como la mayoría de las aplicaciones de libros electrónicos, también puedes pasar páginas tocando o deslizando la pantalla.

Formato de libro electrónico.

  • Sólo para libros de formato fijo. Por ejemplo, libros de papel convertidos directamente en libros de papel o cómics.
  • Admite la importación de archivos PDF y ZIP.

Casos prácticos

  • Lectura general
  • Tocar un instrumento musical mientras se consultan partituras
  • Cocinar mientras se consulta un libro de recetas
  • Leer en trenes y autobuses
  • Lectura para personas con discapacidad física

Dispositivos de apoyo

Utiliza el framework de Apple ARKit/FaceTrackingAPI para la detección de parpadeos. Esto requiere una cámara TrueDepth o un chip A12Bionic o posterior.
