LockInNote

Приложение для записи заметок, ориентированное на виджеты!

Обзор.

Это приложение для ведения заметок, ориентированное на виджеты, для iPhone/iPad/Mac/Apple Watch/Apple Vision Pro.

Вы можете быстро писать заметки с виджета, когда хотите сделать заметку, а затем легко проверить заметку на виджете в любое время!

Кроме того, дизайн заметок на виджете можно гибко настраивать, поэтому его можно использовать для самых разных целей.

Поддержка нескольких платформ

  • iPhone: главный экран, экран блокировки, режим ожидания
  • iPad: главный экран, экран блокировки
  • Mac: центр уведомлений, рабочий стол
  • Apple Watch: сложности, умный стек
  • Apple Vision Pro: виджеты
