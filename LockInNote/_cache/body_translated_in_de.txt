<h2 id="概要"> Überblick.</h2>
<p> Dies ist eine Widget-spezifische Notizen-App für iPhone/iPad/Mac/Apple Watch/Apple Vision Pro.</p>
<p> Sie können schnell Notizen über das Widget schreiben, wenn Sie eine Notiz machen wollen, und dann die Notiz auf dem Widget jederzeit überprüfen!</p>
<p> Darüber hinaus kann das Design der Notizen auf dem Widget flexibel angepasst werden, so dass es für eine Vielzahl von Zwecken verwendet werden kann.</p>
<h3 id="マルチプラットフォーム対応"> Multiplattform-Unterstützung</h3>
<ul>
<li> iPhone: Startbildschirm, Sperrbildschirm, Standby</li>
<li> iPad: Startbildschirm, Sperrbildschirm</li>
<li> Mac: Benachrichtigungszentrale, Schreibtisch</li>
<li> Apple Watch: Komplikationen, Smart Stack</li>
<li> Apple Vision Pro: Widgets</li>
</ul>
